# FractionalCPO Brand Core

## Colors
| Name | Hex | CSS Variable | Usage |
|------|-----|-------------|-------|
| White | #FFFFFF | --white | Cards, content areas |
| Off-white | #F8F8F6 | --off | Primary slide/page background |
| Navy | #002178 | --navy | Brand color, heading accents, sidebar |
| Accent Blue | #0047FF | --accent | Sparingly — tags, highlights |
| Black | rgba(0,0,0,0.83) | --dark | Primary text |
| Muted | rgba(0,0,0,0.5) | --muted | Body text, descriptions |
| Border | #E2E2E2 | --border | Dividers, card borders |

## Typography
- **Font:** Manrope (Google Fonts)
- **Weights:** 300 (light), 400 (regular), 600 (semibold), 700 (bold), 800 (extra-bold)
- **"Skip a weight" rule:** Pair 300 with 700, or 400 with 800. Never adjacent weights.

## Logo
- Text-based: "Fractional" (weight 300) + "CPO" (weight 800)
- On light backgrounds: navy color
- On dark backgrounds: white color
- Position: top-left corner
- No image logo in decks — text only for reliability

## Design Principles
- **Sharp corners** everywhere (no border-radius)
- **Navy accent on 1-2 key words per heading** — the most important concept, not decorative
- **Generous whitespace** — content should breathe
- **Left-aligned text** — never center-align body content
- **No logo strips** with company name text — use actual logos or nothing
- **No prominent navigation** — minimal, text-only where needed
